# NumPy Data Lab

Ce projet regroupe une série d'exercices pratiques visant à maîtriser les bases et concepts intermédiaires de NumPy, bibliothèque fondamentale pour la manipulation de données en Python.

## Objectif

Renforcer les compétences fondamentales en :

- Manipulation de tableaux NumPy
- Indexation et slicing
- Création de matrices 2D et 3D
- Génération de nombres aléatoires
- Opérations vectorisées
- Statistiques de base
- Transformation et normalisation des données

Ce notebook constitue un laboratoire d'entraînement pour les futurs projets en Data Analysis et Data Engineering.

## Contenu du notebook

1. Import des librairies  
   - numpy  
   - pandas (comparaison)

2. Manipulation de tableaux  
   - Création d’arrays 1D  
   - Indexation  
   - Slicing  
   - Modification de valeurs  

3. Tableaux multidimensionnels  
   - Création de matrices 2D  
   - Utilisation de `.shape`  
   - Accès à des éléments spécifiques  

4. Génération aléatoire  
   - Matrices 3D  
   - np.random.random()  

5. Exercices pratiques  
   - Création d’un gradient avec `linspace`  
   - Manipulation conditionnelle  
   - Transformations mathématiques  

## Technologies utilisées

- Python 3.x
- NumPy
- Jupyter Notebook
