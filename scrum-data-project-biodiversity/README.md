# Climate Impact on Biodiversity – Agile SCRUM Project

## Contexte
Projet réalisé dans le cadre d’un module Agile SCRUM.
Analyse de l’impact du changement climatique sur la biodiversité mondiale.

## Objectifs
- Étudier l’évolution des espèces
- Identifier les zones critiques
- Produire des visualisations pertinentes
- Appliquer la méthodologie SCRUM

## Organisation en Sprints
Sprint 0 – Cadrage  
Sprint 1 – Exploration et Nettoyage  
Sprint 2 – Enrichissement  
Sprint 3 – Analyse et Visualisation  
Sprint 4 – Synthèse  

## Outils
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Jupyter Notebook  

## Installation
pip install -r requirements.txt  
jupyter notebook
